## Main Features
- 코스피, 코스닥 종목 검색
- 원하는 기간 내 종가, EPS, BPS 등 금융 데이터 수집: 네이버 금융, [fnspace](https://www.fnspace.com/DataMart/MartList)
- 수집 완료 데이터 로컬 DB 구축
- 데이터 그래프 표시
